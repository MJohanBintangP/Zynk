# 🚀 Fullstack Web App

Ini adalah proyek fullstack web app saya untuk keperluan portofolio,
di web ini saya membuat portal berita yang API nya saya ambil dari CNN

## 📦 Tech Stack

### Frontend

- ⚛️ React.js
- 💨 Tailwind CSS
- 🌐 Fetch API
- 🌍 vercel (Deployment)
- 🔧 Vite (Build tool)

### Backend

- 🛠️ node.js
- 🔌 REST API
- 🌍 Railway (Deployment)

---

## 🚀 Cara Menjalankan Proyek

### 🔧 Jalankan Frontend

```bash
cd frontend
npm install -g pnpm (if you use pnpm)
npm/pnpm install
npm/pnpm run dev
```

### 🔧 Jalankan Backend

```bash
cd backend
npm install
node index.js
```
